## A propos de l'application

Projet full stack basé sous Laravel 8 avec Tailwind CSS, Livewire, Alpine.js. 

Il s'agit d'une application de jeu vidéo qui extrait les données de l'API IGDB de Twitch et affiche des informations sur le jeu telles que les titres populaires, à venir, des vidéos et des captures d'écran.
